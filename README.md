  <br>
</p>

&nbsp;

#### install

```
npm i travis.yml
```

#### usage

```
npx travis.yml
```

&nbsp;

this creates a `.travis.yml` in your working directory

```
language: node_js
node_js:
  - "8"
cache:
  directories:
    - node_modules
notifications:
  email: false
```
